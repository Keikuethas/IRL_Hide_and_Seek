# Что это?
Это мобильное приложение для Android, которое помогает с организацией подвижных игр в реальной жизни.

Создаётся в рамках ознакомительной практики и обучения мобильной разработке двумя студентами РТУ МИРЭА.
Работоспособная версия будет готова к маю 2026.

Приведённое описание, скорее всего, изменится.
# Как это работает?
1. Все игроки скачивают приложение.
2. Кто-то создаёт комнату и настраивает игру.
3. Остальные игроки подключаются и распределяют роли.
4. Много веселья!
# Нет, серьёзно, *как* это работает?
Ну, хорошо. Пройдёмся по фишкам:
- Отслеживание местоположения игрока
- Ограничение игровой зоны
- Активные способности
